file Organizer

ATTENTION

#####################

:: Ceci est un commentaire d'une ligne d'un fichier bash
:: Ce fichier sert à organiser vos fichiers
:: par example:
:: (document text, musique, images/photos/vidéos, logiciels..)
:: Du Dossier Dans Des Dossiers Nommées Par Extentions
:: (.txt .jpg .png .mp3 .mp4 .swf .xml .html .css .js .json etc..).
:: Il vous sera plus facile par la suite de supprimer les doublons
:: Sauvegarder dans le dossier où vous souhaitez organiser vos fichiers
:: NE PAS EXECUTER CE FICHIER DANS LE DOSSIERS "C:\WINDOWS\SYSTEM32"
:: ENJOY!!
:: Pour plus d'information à propos de ce script de type batch (bash.sh.bat.cmd) Visiter: https://www.ss64.com
:: Code Start Here ...
::
for %%a in (".\*") do (
if "%%~xa" NEQ "" if "%%~dpxa" NEQ "%~dpz0" (
if not exist "%%~xa" mkdir "%%~xa"
move "%%a" "%%~dpa%%~xa\"
));
